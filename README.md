## Projets web

Ce dépôt comporte quatre sites web amusants complétés dans un cadre scolaire. En plus d'une base HTML et CSS, la plupart des exemples font usages de FlexBox, media queries, JS et DOM, typiques pour le développement de sites web.

#### Liste des projets
- Labo1_Site_Simple: usage de base de FlexBox
- Labo2_Google: reproduction partielle de la page d'accueil de Google.
- Labo3_Pixel_Art: grille de pixels colorable, de résolution variable. L'usager peut se faire une palette de couleurs, en ajoutant plusieurs de son choix.
- Labo4_Giphy: Démonstration de l'usage d'un API web nommé "Giphy" pour obtenir et afficher des images GIF ou bien des autocollants ("stickers") en fonction de ce que l'usager tape dans un boîte de recherche. (requiert une clef pour l'API)
